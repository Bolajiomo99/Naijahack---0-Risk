Customization

Customize the User Interface

For a session you can change the background color and choose dark or light text. This affects the PIN pad and processing screen:


Customize Accessibility

If you wish to change the text that appears when a user goes to the Accessibility Settings screen you can do so by defining the strings access_service_summary and access_service_descript in your strings.xml file. access_service_descript is the one that users will see on the settings page.

Translate

You can translate the strings that are part of Maths Clinic's UI components. The list of string resources to translate are in the root folder of the Maths Clinic SDK's aar packaging called public.txt you can find the current english value of those strings in /res/values/values.xml
